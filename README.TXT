# Persyaratan System
- SLiMS 3 Stable 15 "Matoa" (download di http://slims.web.id/)
- Modul Plugins aktif versi 3.15-2.x "Nantu"
  (download di http://sutriadi.web.id/)

# Instalasi
- uraikan paket instalasi scrop-x.x-x.x-x.tar.gz di dalam direktori
  ./admin/modules/plugins/include/ sehingga akan menghasilkan direktori
  ./admin/modules/plugins/includes/scrop/
- masuk ke menu Plugins, kemudian aktifkan plugin SCrop
catatan: bila melakukan instalasi dari paket yang diambil dari github,
mohon untuk mengubah nama direktori hasil penguraian
dari ./sutriadi-scrop-xxx menjadi ./scrop

# Hapus Instalasi
- masuk ke menu Plugins, kemudian non-aktifkan plugin SCrop
- hapus item scrop dari submenu dataTables bila perlu
- hapus direktori scrop dari direktori
  ./admin/modules/plugins/include/ bila perlu

# Menjalankan Plugin
- masuk ke menu Plugins
- klik tombol Launch pada baris plugin aktif bernama SCrop
