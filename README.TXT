-- Persyaratan System
- SLiMS 3 Stable 14 "Seulanga"
- Modul Plugins aktif versi 0.3 patch 1 (download di http://sutriadi.web.id/)

-- Instalasi
- uraikan paket instalasi scrop-x.x.x.tar.gz di dalam direktori ./admin/modules/plugins/
  sehingga akan menghasilkan direktori ./admin/modules/plugins/scrop/
- masuk ke menu Plugins, kemudian aktifkan plugin SCrop

-- Hapus Instalasi
- masuk ke menu Plugins, kemudian non-aktifkan plugin SCrop
- hapus direktori smember dari direktori ./admin/modules/plugins/ bila perlu

-- Menjalankan Plugin
- masuk ke menu Plugins
- klik link Launch pada baris plugin aktif bernama SCrop
